# Смета-Стройка AI Калькулятор

Фронтенд-прототип AI-инструмента для строительных смет, планировок, тендеров и коммерческих предложений.

## Быстрый старт

```bash
npm install
npm run dev
```

После запуска открой локальный адрес, который покажет Vite.

## Что внутри

- `index.html` - точка входа приложения.
- `app.jsx` - корневой React-компонент.
- `screen-planning.jsx`, `planning-*.jsx` - экран планировки, слои, расчеты и панели.
- `screens-rest.jsx` - остальные разделы: сметы, КП, тендеры, прайсы, настройки.
- `styles.css` - основные стили интерфейса.
- `screenshots/` - контрольные скриншоты прототипа.
- `uploads/` - тестовые загруженные изображения.

## Текущий стек

Проект пока работает как статический прототип: React и Babel подключены через CDN в `index.html`. Vite используется как удобный локальный dev-сервер.

Практически это значит: старт разработки быстрый, но для production лучше следующим шагом перевести JSX в нормальные ES-модули и подключить React через npm.

## Ближайшие технические шаги

1. Перевести глобальные JSX-файлы на модульную структуру `src/`.
2. Подключить React через npm вместо CDN.
3. Разделить большой код по доменам: планировка, сметы, КП, тендеры, прайс-листы.
4. Добавить базовую проверку качества: lint, форматирование и smoke-тест запуска.
